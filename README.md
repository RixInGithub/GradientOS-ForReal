# GradientOS V2 "For Real"
**************************
This is the WebAssembly remake of GradientOS.
Instead of using the famous slacker of WASM compilers Emscripten, I'm using a shit ton of `clang`, recycled code, ideas, hopes and dreams.

*Oh and also `wasm-ld` apparently heheh*

## Building
***********

Just run the `./build` command on Linux/Unix/GNU/whatever the fuck and watch the magic happen right before your eyes!
